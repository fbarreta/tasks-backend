## Description

Backend Application for task management.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# The application should run by default at http://localhost:3000
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
